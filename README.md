# Myntra Clone

A polished Myntra-inspired React shopping experience built for portfolio and resume showcase.

## Features

- Data-driven product catalog rendered from `src/data/catalog.js`
- Search across product name, brand, category and tags
- Category filters for Men, Women, Kids, Home and Beauty
- Sorting by recommended, price, rating and discount
- Wishlist save and remove flow
- Login and signup modal with profile state
- Shopping bag drawer with item removal, delivery address, coupon card and price summary
- Brand deal cards, hero banners and category shortcuts
- Lucide icon system for header actions, product cards and checkout UI
- Responsive layout for desktop, tablet and mobile
- Local deploy-safe images in `public/images`

## Tech Stack

- React
- Vite
- Lucide React
- CSS Grid and Flexbox

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
